
# PlantUML

Install PlantUML and supporting dependencies to enable PlantUML VS Code extension, enhanced markdown previews and the
Pandoc PlantUML filter.

## Example Usage

```json
"features": {
        "ghcr.io/danlewisao/devcontainer-features/ao.ade.plantuml": {
            "version": "latest"
        }
}
```
