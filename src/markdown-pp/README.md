
# Copier

Install the [Copier](https://github.com/copier-org/copier) templating engine to allow quick and easy provisioning of
architecture as code documents.

## Example Usage

```json
"features": {
        "ghcr.io/danlewisao/devcontainer-features/ao.ade.copier": {
            "version": "latest"
        }
}
```
